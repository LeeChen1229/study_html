# Code Wiki — 机器学习课程可视化插图项目

## 1. 项目概述

本项目是一套**机器学习/深度学习课程可视化插图集合**，由 20 个独立的 HTML 文件组成。每个文件对应一个 ML/DL 核心主题，以 **3:4 比例的"海报卡片"（Poster Notes）** 形式呈现知识点，包含概念说明、公式展示、Canvas 动态图表和流程图解。

项目特点：
- **零依赖**：纯 HTML + CSS + JavaScript，无需任何框架或构建工具
- **自包含**：每个 HTML 文件完全独立，可直接在浏览器中打开
- **响应式**：支持桌面端和移动端自适应布局
- **高保真渲染**：Canvas 绑定 `devicePixelRatio`，支持 Retina 屏幕清晰渲染

## 6. JavaScript 绘图系统

### 6.1 核心工具函数

#### `setupCanvas(canvas)`

初始化 Canvas，适配高分屏。

```javascript
function setupCanvas(canvas) {
  const ratio = window.devicePixelRatio || 1;
  const rect = canvas.getBoundingClientRect();
  canvas.width = Math.floor(rect.width * ratio);
  canvas.height = Math.floor(rect.height * ratio);
  const ctx = canvas.getContext("2d");
  ctx.scale(ratio, ratio);
  return { ctx, width: rect.width, height: rect.height };
}
```

- **输入**：Canvas DOM 元素
- **输出**：`{ ctx, width, height }` — 2D 上下文及逻辑尺寸
- **关键**：通过 `devicePixelRatio` 缩放确保 Retina 屏清晰渲染

#### `drawGrid(ctx, width, height, margin)`

绘制坐标网格背景。

- 绘制 6×4 的浅色网格线
- 绘制坐标轴（稍深色）
- 参数 `margin` 控制绘图区域边距：`{ top, right, bottom, left }`

#### `drawText(ctx, text, x, y, color)`

在 Canvas 上绘制文字标注。

- 默认颜色 `#5f6670`
- 字体 `13px Segoe UI`

#### `mapPoint(u, v, width, height, margin)`

将归一化坐标 `[0,1]×[0,1]` 映射为 Canvas 像素坐标。

- `u` → 水平方向（0=左, 1=右）
- `v` → 垂直方向（0=底, 1=顶，数学坐标系）
- 返回 `{ x, y }`

### 6.3 渲染入口

所有文件统一使用以下模式触发渲染：

```javascript
function renderAll() {
  drawChartA();
  drawChartB();
  // ...
}

window.addEventListener("load", renderAll);
window.addEventListener("resize", renderAll);
```

- **`load` 事件**：页面加载完成后首次渲染
- **`resize` 事件**：窗口大小变化时重新渲染（Canvas 尺寸自适应）

## 11. 设计规范总结

### 视觉风格

- **纸张质感**：米黄底色 + 网格线 + 毛玻璃卡片，模拟纸质笔记
- **配色**：蓝/紫（主） + 琥珀（辅） + 翠绿（正面） + 砖红（警示）
- **字体**：标题用衬线体（Georgia/宋体），正文用无衬线体（Segoe UI/微软雅黑）
- **圆角**：卡片 18px，页面容器 30px，标签 999px（胶囊形）

### 内容规范

- 每个主题 3 张卡片，3:4 比例
- Card 01：概念 + 流程 + 公式 + 步骤 + 一句话记忆
- Card 02：Canvas 图表 + 解释
- Card 03：深入/对比 + 总结
- 文案风格：口语化、入门友好，避免公式推导，强调直觉理解
