自相关函数的典型形态

自相关函数用于度量信号与其延时版本之间的相似程度。对信号分析而言，它不仅能描述 “相关持续多久”，还能揭示带宽、载频以及周期结构。下面这张课程插图将宽带信号、 窄带信号和周期性信号放在同一个观察框架下，直接比较它们的自相关形状差异。

定义 Rxx(τ) = E[x(t)x(t+τ)]
通性 Rxx(τ) = Rxx(-τ)
峰值 Rxx(0) 最大

怎么看这张图

先看时域波形，再看对应的自相关曲线。自相关越窄，说明信号相关时间越短，通常频带越宽；自相关越缓慢振荡，常对应更强的窄带或载频特征。

课程结论

宽带看“尖峰”，窄带看“振荡包络”，周期信号看“重复峰列”。这三种图形是课程、考试和工程判断中最常见的自相关模式。

从自相关反推信号特性

自相关函数可以理解成“延时后还像不像自己”。当信号中高频和多频成分丰富时，稍微延时一点波形就完全变了样，相似性会很快消失；当信号含有明显载频或稳定周期时，即使延时了若干个周期，波形仍会和自身重合，相似性就会以振荡或重复峰值的形式保留下来。所以看自相关的形状，就能反推出信号的频域特征。

宽带信号 自相关主峰很窄，离开零时延后迅速衰减，说明相关时间短、频谱展开宽。比如白噪声或宽带脉冲，时域上变化剧烈，稍微延时就不像自己了，所以自相关只在原点附近有一个尖峰。频域上则对应很宽的功率谱——这就是“时窄频宽”的体现。
窄带信号 自相关呈余弦样振荡，并带有缓慢衰减包络，说明围绕中心频率的相关性较强。比如一个带通滤波后的信号，延时若干个载频周期后仍能和自身对齐，所以自相关会来回振荡。振荡频率大致对应中心频率，包络宽度则反映带宽——包络衰减越慢，说明频带越窄。
周期性信号 自相关会重复出现等间隔峰值，峰间距就是原信号周期或其整数倍。比如一个正弦波，延时一个整周期后波形完全重合，自相关就出现一个峰值；延时两个周期又重合，再出现一个峰值。这是识别周期信号最直观的方法——如果自相关图上出现等间距的峰列，直接读间距就能得到周期。
延时轴 τ 负延时 ← 0 → 正延时
宽带: 主峰窄 窄带: 振荡持续 周期: 峰列重复

自相关的几个核心性质

除去具体形状差异，三类信号都满足一些基础规律。这些规律能帮助你判断图画得对不对，也能作为推导和考试中的快速检查点。比如看到自相关在 τ=0 处不是最大值，那图一定画错了；看到自相关左右不对称，也一定有问题。记住这几个性质，可以快速排除常见错误。

偶对称 自相关函数关于 τ = 0 对称，因此左半边和右半边形状应镜像一致。这从定义就能理解：把信号往左延时 τ 和往右延时 τ，得到的相似程度是一样的。所以画自相关图时，只需要画右半边，左半边镜像过去就行。
零延时最大 τ = 0 时与自身完全重合，所以相关值最大，通常是图中的最高点。任何非零延时都不可能比完全不延时更“像自己”，因此 Rxx(0) 是全局最大值。这个性质在考试中经常用来判断自相关图是否画正确——如果最高点不在 τ=0 处，那一定有问题。
周期保持 若原信号具有周期 T，则自相关也具有周期 T，这是识别周期信号最直接的图形证据。因为延时 T 之后信号和自身完全一样，延时 2T 又完全一样，所以自相关会在 T、2T、3T……处重复出现峰值。反过来，如果自相关图上出现等间距峰列，你就可以直接读出间距，那就是信号的周期。

宽带

主峰尖窄

宽带信号含有更丰富的频率分量，时域细节变化快，因此稍微延时后就不再像原信号，自相关只在零时延附近显著；在频域中则表现为更宽的能量铺展。

时域示意 x(t)细节密集 / 快速变化
自相关 Rxx(τ)窄主峰 / 快速衰减
频域示意 Sxx(f)带宽较宽 / 连续展开
教学提示 相关时间短，对应频域更宽。这是“时宽频窄 / 时窄频宽”关系在自相关视角下的直观体现。

窄带

振荡包络

窄带信号通常围绕某个中心频率振荡。延时后仍保留明显的相位关系，因此自相关常呈现余弦样振荡，并伴随较慢衰减的包络；在频域中能量集中在较窄频带内。

时域示意 x(t)载频明显 / 振荡稳定
自相关 Rxx(τ)振荡对称 / 衰减较慢
频域示意 Sxx(f)中心频率明显 / 频带较窄
教学提示 自相关中的振荡频率通常对应中心载频信息，包络宽度则反映带宽大小。

周期性

重复峰列

对于周期信号，只要延时等于一个周期或其整数倍，信号仍会与自身重新对齐，因此自相关会周期性重复出现峰值；在频域中则常对应离散谱线或谐波结构。

时域示意 x(t)重复结构 / 周期 T
自相关 Rxx(τ)峰值间隔 = T
频域示意 Sxx(f)离散谱线 / 谐波成分
教学提示 如果图上出现等间隔峰列，可以优先判断原信号具有明显周期结构。
